Plugin that displays a form to signup e-mails to selectecd list in Mailchimp

== Description ==
This plugin displays a simple form so that your users can subscribe to a selected list from Mailchimp.


WHAT IS MAILCHIMP?
Mailchimp is a newsletter service that allows you to send out email campaigns to a list of email subscribers. It is free for lists up to 2000 subscribers, which is why it is the newsletter-service of choice for thousands of businesses.

This plugin allwos you to have your visitor sign up to a selected list in your Mailchimp account.

This plugin is dependent on Mailchimp: https://mailchimp.com/legal/
If you are not yet using Mailchimp you can create an account here: https://login.mailchimp.com/signup/.

== Installation ==

e.g.

1. Upload folder `super-mailchimp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [mailchimp]
